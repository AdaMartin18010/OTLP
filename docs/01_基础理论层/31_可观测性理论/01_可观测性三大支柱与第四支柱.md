# 可观测性三大支柱与第四支柱

> **最后更新**: 2026-03-17

---

## 1. 三大支柱

### Metrics（指标）

- **特点**: 聚合性、时序性、低开销
- **类型**: Counter, Gauge, Histogram, Summary
- **场景**: 系统健康度、容量规划

### Logs（日志）

- **特点**: 详细性、文本性、灵活性
- **级别**: ERROR, WARN, INFO, DEBUG, TRACE
- **场景**: 错误排查、审计追踪

### Traces（追踪）

- **特点**: 请求链路、因果关系、分布式
- **核心**: Trace, Span, SpanContext
- **场景**: 延迟分析、故障定位

## 2. 第四支柱：Profiles

- **Continuous Profiling**: 持续自动采集
- **类型**: CPU, Memory, Goroutine, Lock, I/O
- **开销**: <5% CPU
