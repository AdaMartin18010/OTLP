## 🎯 一、项目主题体系定义

### 1.1 主主题（核心定位）

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │

│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 十二大子主题（核心内容架构）

| 编号 | 子主题 | 核心内容 | 相关目录/文件 |
|------|--------|----------|---------------|
| **T1** | **理论基础** | 数学基础、形式化验证、证明策略 | `docs/01_理论基础/`, `docs/02_THEORETICAL_FRAMEWORK/` |
| **T2** | **标准规范** | OTLP协议、语义约定、规范详解 | `docs/01_OTLP核心协议/`, `docs/02_Semantic_Conventions/` |
| **T3** | **数据模型** | Traces、Metrics、Logs、Baggage、Resource | `docs/03_数据模型/` |
| **T4** | **传输协议** | gRPC、HTTP、Protocol Buffers编码 | `docs/01_OTLP核心协议/` |
| **T5** | **核心组件** | SDK、Collector、Context Propagation | `docs/04_核心组件/` |
| **T6** | **采样与性能** | 采样策略、性能优化、压缩技术 | `docs/05_采样与性能/` |
| **T7** | **部署运维** | Docker、K8s、安全、监控、故障排查 | 根目录部署指南系列 |
| **T8** | **实战案例** | 各行业可观测性实战案例 | `docs/06_实战案例/` |
| **T9** | **学术研究** | 论文框架、形式化证明、案例研究 | `academic/` (压缩包) |
| **T10** | **前沿技术** | eBPF、GenAI、Wasm、Arrow、边缘计算 | `docs/12_前沿技术/` |
| **T11** | **知识中心** | 概念索引、知识图谱、矩阵、导图、术语 | `00_知识中心/` |
| **T12** | **工具链** | 配置生成器、验证工具、自动化工具 | `docs/工具/`, `scripts/`, `tools/` |
