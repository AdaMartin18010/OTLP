# 范围-权威对齐矩阵

> **用途**: 按本项目三条内容范围，对齐网络最新权威来源，驱动「缺哪补哪」的文档与示例更新。
> **更新节奏**: 与 [批判性评价与持续改进计划](README.md) 季度/月度评审同步。
> **最后更新**: 2025-02-10

---

## 范围界定

| 范围项 | 定义要点 | 应用场景 |
|--------|----------|----------|
| **OpenTelemetry OTLP** | trace/metric/log 传输、序列化（gRPC/HTTP）、语义约定 | 可观测性、微服务遥测 |
| **OTLP Metrics Data Model** | 支持 pre-aggregated 时序数据、与 Prometheus/StatsD 双向映射 | 监控、指标收集 |
| **OTLP Logs Data Model** | 统一日志结构（Event、属性、资源）、多种日志格式无歧义映射 | 日志聚合、审计 |

## 使用说明

- **驱动更新**: 当「缺口」列有项时，优先在对应「本项目文档位置」补全，并在「当前版本/日期」中更新版本信息。
- **思维表征**: 新增或更新导图/矩阵/树图后，在「思维表征文档」列补充链接。
- **季度评审**: 每季度对照权威来源检查矩阵，更新缺口与版本，并同步 [02_批判性评价/范围专项批判性评价](02_批判性评价/范围专项批判性评价.md) 与 [03_改进计划](03_改进计划/)。
