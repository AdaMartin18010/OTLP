## 📚 文档架构 (7层模型)

本知识库采用7层架构组织，对标最新权威标准：

```
├── 00_参考资料/           # 原始规范、RFC、学术论文
├── 01_基础理论层/         # 核心理论、形式化论证、数学模型
├── 02_核心协议层/         # OTLP协议、语义约定、编码规范
├── 03_核心实现层/         # SDK实现、Collector实现、代码示例
├── 04_部署运维层/         # 部署架构、性能调优、监控告警
├── 05_应用实践层/         # 行业案例、最佳实践、场景方案
├── 06_工具生态层/         # 工具使用、集成方案、扩展开发
├── 07_项目管理层/         # 治理规范、合规检查、发布管理
└── 99_归档/               # 历史版本、废弃文档
```
