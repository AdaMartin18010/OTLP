### 1.1 GenAI语义约定 - Agent相关属性 (Draft)

**状态**: Draft (草案阶段)
**来源**: 基于Google AI Agent白皮书

#### 新增Agent属性

```yaml
# Agent基础属性
gen_ai.agent.id:              # Agent唯一标识
gen_ai.agent.name:            # Agent名称
gen_ai.agent.version:         # Agent版本

# Agent调用属性
gen_ai.agent.invocation.id:   # 调用ID
gen_ai.agent.invocation.type: # 调用类型
  - synchronous   # 同步调用
  - asynchronous  # 异步调用
  - streaming     # 流式调用

# Agent系统属性
gen_ai.agent.system:          # Agent框架/系统
  - langchain
  - llamaindex
  - autogen
  - crewai
  - custom
```

#### Agent Span示例

```python
from opentelemetry import trace

tracer = trace.get_tracer(__name__)

# Agent创建Span
with tracer.start_as_current_span("agent.create") as span:
    span.set_attribute("gen_ai.agent.id", "agent-001")
    span.set_attribute("gen_ai.agent.name", "CustomerServiceAgent")
    span.set_attribute("gen_ai.agent.version", "1.0.0")
    span.set_attribute("gen_ai.agent.system", "langchain")

# Agent调用Span
with tracer.start_as_current_span("agent.invoke") as span:
    span.set_attribute("gen_ai.agent.id", "agent-001")
    span.set_attribute("gen_ai.agent.invocation.id", "inv-12345")
    span.set_attribute("gen_ai.agent.invocation.type", "synchronous")
```

### 1.2 GenAI - Cost Tracking增强

**状态**: 实验性增强

#### 新增Cost属性

```yaml
# 成本追踪 (实验性)
gen_ai.usage.cost.currency:   # 货币代码 (USD, CNY, EUR)
gen_ai.usage.cost.amount:     # 成本金额
gen_ai.usage.cost.model_price_input_per_token:   # 输入Token单价
gen_ai.usage.cost.model_price_output_per_token:  # 输出Token单价
```

#### 成本计算示例

```python
# 成本计算示例
input_tokens = 150
output_tokens = 200
input_price = 0.00001  # $0.01 per 1K tokens
output_price = 0.00003  # $0.03 per 1K tokens

input_cost = input_tokens * input_price
output_cost = output_tokens * output_price
total_cost = input_cost + output_cost

span.set_attribute("gen_ai.usage.cost.currency", "USD")
span.set_attribute("gen_ai.usage.cost.amount", total_cost)
span.set_attribute("gen_ai.usage.cost.model_price_input_per_token", input_price)
span.set_attribute("gen_ai.usage.cost.model_price_output_per_token", output_price)
```

## 二、稳定性变更

### 2.1 升级到Stable的约定

| 约定领域 | v1.39状态 | v1.40状态 | 说明 |
|:---------|:---------:|:---------:|:-----|
| HTTP Client/Server | Stable | Stable | 无变化 |
| Database | Stable | Stable | 无变化 |
| GenAI - LLM spans | Development | Development | 持续开发 |
| GenAI - Agent spans | - | **Draft** | 新增草案 |
| System | Stable | Stable | 无变化 |
| Process | Stable | **Stable+** | 新增属性 |

### 2.2 废弃属性

| 属性 | 状态 | 说明 |
|:-----|:----:|:-----|
| `gen_ai.prompt` | 敏感数据 | 建议使用span event |
| `gen_ai.completion` | 敏感数据 | 建议使用span event |

## 三、迁移指南
