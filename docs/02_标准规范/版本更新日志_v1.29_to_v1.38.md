## ⚠ 三、重大变更

### 3.1 属性重命名 (破坏性变更)

| 旧属性名 | 新属性名 | 版本 | 影响 |
|:---|:---|:---:|:---:|
| `http.status_code` | `http.response.status_code` | v1.30 | 中 |
| `http.method` | `http.request.method` | v1.30 | 中 |
| `http.url` | `http.request.url` (不变) | - | - |
| `db.system` | `database.system` | v1.35 | 高 |
| `db.operation` | `database.operation` | v1.35 | 高 |
| `messaging.destination` | `messaging.destination.name` | v1.36 | 中 |
| `rpc.method` | `rpc.method.name` | v1.34 | 低 |

**迁移指南**:

```python
# 旧代码
span.set_attribute("http.status_code", 200)
span.set_attribute("db.system", "postgresql")

# 新代码
span.set_attribute("http.response.status_code", 200)
span.set_attribute("database.system", "postgresql")
```

---

### 3.2 属性值变更

#### HTTP状态码分类

```yaml
# v1.32.0+ 新增状态码类别
http.response.status_code: 200
http.response.status_code_class: 2xx  # 新增

# 类别定义
1xx: informational
2xx: success
3xx: redirection
4xx: client_error
5xx: server_error
```
