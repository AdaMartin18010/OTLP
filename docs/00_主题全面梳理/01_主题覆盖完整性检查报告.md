### 1.2 主题编号规则

#### 编号格式

```text
主题编号规则:
  格式: T{layer}.{major}.{minor}.{sub}

  示例:
    T1.1.1: 理论基础层.数学基础.集合论基础
    T2.3.2.1: 协议规范层.数据模型.Metrics模型.计数器类型
    T5.1.2: 前沿技术层.AI驱动运维.AI日志分析
```

#### 编号验证规则

```haskell
-- 主题编号验证
validateTopicNumber :: TopicNumber -> Bool
validateTopicNumber num =
  validLayer num.layer (1 <= num.layer && num.layer <= 6) &&
  validMajor num.major (num.major >= 1) &&
  validMinor num.minor (maybe True (>= 1) num.minor) &&
  validSub num.sub (maybe True (>= 1) num.sub)
```

### 1.3 主题状态定义

#### 状态类型

```haskell
-- 主题状态
data TopicStatus
  = Planned      -- 计划中 (0-20%)
  | InProgress   -- 进行中 (21-80%)
  | Completed    -- 已完成 (81-100%)
  | Deprecated   -- 已废弃
  | NeedsUpdate  -- 需要更新

-- 状态转换规则
statusTransition :: TopicStatus -> TopicStatus -> Bool
statusTransition Planned InProgress = True
statusTransition InProgress Completed = True
statusTransition Completed NeedsUpdate = True
statusTransition _ Deprecated = True
statusTransition _ _ = False
```

  , overallCoverage = calculateOverallCoverage project

-- 主题覆盖率
calculateTopicCoverage :: Project -> Float
calculateTopicCoverage project =
  let totalTopics = countTotalTopics project
      completedTopics = countCompletedTopics project
  in fromIntegral completedTopics / fromIntegral totalTopics
