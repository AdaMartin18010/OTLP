## 部署架构

### 部署模式

| 模式 | 特点 | 适用场景 |
|------|------|----------|
| Agent | 与应用同主机 | 元数据丰富、早期处理 |
| Gateway | 集中式 | 统一管理、减少出口 |
| 混合 | Agent + Gateway | 大规模生产推荐 |

### 平台支持

- Kubernetes (DaemonSet/Deployment)
- Docker/Swarm
- VM/Bare Metal
- Serverless

## 性能优化

### 采样优化

- Head采样: 减少90%数据
- Tail采样: 智能保留错误/慢请求

### 批处理优化

| 场景 | 配置 | 效果 |
|------|------|------|
| 低延迟 | 100ms/256 | <1ms延迟 |
| 平衡 | 1s/1024 | 推荐配置 |
