## 🎯 使用指南

### 按目标选择

```
了解全貌 → 01_思维导图/
技术选型 → 02_决策树/技术选型决策树.md
问题诊断 → 02_决策树/故障排查决策树.md
方案对比 → 03_多维矩阵/
```

### 推荐路径

**初学者**: 主题总览思维导图 → 技术选型决策树 → 语言SDK对比矩阵

**技术人员**: 技术选型决策树 → 故障排查决策树 → Collector组件矩阵

**架构师**: 后端方案对比矩阵 → Collector组件矩阵 → 主题关系图谱
