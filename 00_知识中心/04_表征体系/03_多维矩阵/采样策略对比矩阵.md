## 🎯 场景推荐矩阵

### 按流量规模推荐

| 流量规模 | 推荐策略 | 配置建议 | 成本节约 |
|:---|:---:|:---|:---:|
| **<100 TPS** | AlwaysOn | 全量采集 | 0% |
| **100-1K TPS** | Probabilistic 50% | trace_id_ratio_based: 0.5 | 50% |
| **1K-10K TPS** | ParentBased | parent_based: root=probabilistic(0.3) | 70% |
| **10K-100K TPS** | TailBased | 保留error+slow | 85% |
| **>100K TPS** | TailBased + RateLimiting | 复合策略 | 90%+ |

### 按应用场景推荐

| 场景 | 推荐策略 | 原因 |
|:---|:---:|:---|
| **开发/测试** | AlwaysOn | 需要完整数据调试 |
| **性能测试** | AlwaysOn | 需要准确基准 |
| **生产高流量** | TailBased | 保留异常，节约成本 |
| **生产关键路径** | ParentBased | 保证Trace完整性 |
| **成本敏感** | Probabilistic 10% | 平衡成本和覆盖 |
| **合规审计** | AlwaysOn | 满足合规要求 |

### 按后端能力推荐

| 后端 | 推荐策略 | 原因 |
|:---|:---:|:---|
| **低成本后端** | Probabilistic | 后端成本敏感 |
| **高吞吐后端** | TailBased | 能承受突发流量 |
| **实时分析** | ParentBased | 保证实时性 |
| **长期存储** | TailBased | 只存有价值数据 |

## 🛠️ 配置示例矩阵

### SDK配置示例

| 策略 | Java | Go | Python |
|:---|:---|:---|:---|
| **AlwaysOn** | `setSampler(alwaysOn())` | `trace.AlwaysSample()` | `sampler=AlwaysOnSampler()` |
| **Probabilistic** | `setSampler(traceIdRatioBased(0.1))` | `trace.TraceIDRatioBased(0.1)` | `sampler=TraceIdRatioBased(0.1)` |
| **ParentBased** | `setSampler(parentBased(...))` | `trace.ParentBased(...)` | `sampler=ParentBasedSampler(...)` |

### Collector配置示例

| 策略 | Processor配置 |
|:---|:---|
| **TailBased** | `tail_sampling: decision_wait: 10s` |
| **Probabilistic** | `probabilistic_sampler: sampling_percentage: 10` |
