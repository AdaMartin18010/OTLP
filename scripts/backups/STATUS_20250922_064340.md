## 使用指南

### 语义验证

```bash
# 验证语义约定
python governance/semantic-validator.py data.json
```

### 变更提案

1. 使用 `CHANGE_PROPOSAL_TEMPLATE.md` 创建提案
2. 提交到治理委员会审查
3. 通过后更新相关文档

### 合规检查

1. 使用 `COMPLIANCE_CHECKLIST.md` 进行自检
2. 定期审查和更新
3. 记录合规状态
