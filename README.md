## 🛠️ 快速开始

### Docker 一键启动

```bash
# 克隆仓库
git clone <repo-url>
cd OTLP

# 启动完整环境
docker-compose up -d

# 查看追踪
open http://localhost:16686  # Jaeger UI
```

### 代码示例

```bash
# Go示例
cd examples/go && go run .

# Python示例
cd examples/python && pip install -r requirements.txt && python app.py
```
